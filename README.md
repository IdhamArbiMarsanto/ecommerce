# E-Commerce Frontend

## Deskripsi
Proyek ini merupakan aplikasi **frontend e-commerce berbasis web**
yang berfungsi untuk menampilkan katalog produk, proses pemesanan,
serta simulasi checkout pengguna.

Aplikasi frontend ini dikembangkan sebagai bagian dari **tugas proyek
kuliah** dan terintegrasi dengan backend e-commerce yang dikembangkan
dalam repository terpisah.

---

## Tujuan Proyek
- Memenuhi tugas proyek mata kuliah pengembangan aplikasi web
- Mengimplementasikan konsep UI/UX pada sistem e-commerce
- Memisahkan pengembangan frontend dan backend sesuai arsitektur aplikasi modern

---

## Fitur
- Halaman Beranda
- Katalog Produk
- Detail Produk
- Keranjang Belanja
- Checkout
- Login & Register (tampilan)
- Riwayat / Status Pesanan (tampilan)

---

## Teknologi yang Digunakan
- HTML5

- Bootstrap

---

## Tampilan Halaman
- Beranda

- Keranjang
- Checkout

---

## Cara Menjalankan Aplikasi
1. Clone repository frontend ini
2. Buka folder project
3. Jalankan file `index.html` melalui browser  
   atau gunakan ekstensi **Live Server** pada Visual Studio Code

---

## Repository Terkait
Proyek e-commerce ini dikembangkan dengan arsitektur terpisah antara
frontend dan backend untuk memudahkan pengembangan dan pengelolaan sistem.

- Frontend Repository:  
  https://github.com/IdhamArbiMarsanto/ecommerce.git

## Catatan
Proyek ini berfokus pada **pengembangan frontend**.
Pengolahan data dan logika bisnis sepenuhnya ditangani oleh backend.
